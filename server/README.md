# SendAnything Server

Server for SendAnything written in C++ using `websocketpp` and GTK3

## Setup
* Install GTKMM3 using your distro's package manager
* Build using CMake 
* Run the binary

### Dependencies:
* [websocketpp](https://github.com/zaphoyd/websocketpp)
* [Gtkmm 3](https://www.gtkmm.org/en/)
